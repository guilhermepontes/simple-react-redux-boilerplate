#### Whats inside?
- React, Redux, React Router, Redux Logger, Redux Promise Middleware, Redux Thunk

- ESLint, Stylelint
- whatwg-fetch, lodash, bem-classname

#### Installing
`yarn install`

#### Running the application
- Dev: `yarn run start`
- Prod: `yarn run start:prod`

#### Running tests
`yarn run test`

#### Running linters
- All: `yarn run lint`
- CSS: `yarn run lint:css`
- JS: `yarn run lint:js`
